# GalleryDialog — Компонент галереи

## Расположение

```
/components/gallery-dialog.tsx
```

## Описание

**GalleryDialog** — переиспользуемый компонент полноэкранной галереи изображений, построенный на базе Radix UI Dialog.

## Основные возможности

- Полноэкранное отображение (95vw x 90vh)
- Навигация стрелками (кнопки и клавиатура ←/→)
- Точки-индикаторы текущего слайда
- Подписи к изображениям
- Возможность открыть галерею с конкретного изображения (`initialIndex`)
- Кнопка закрытия (крестик или Escape)

## Использование

```tsx
import { GalleryDialog, type GalleryImage } from "@/components/gallery-dialog"

const images: GalleryImage[] = [
  { src: "/path/to/image1.jpg", caption: "Подпись 1" },
  { src: "/path/to/image2.jpg", caption: "Подпись 2" },
]

// Базовое использование
<GalleryDialog
  trigger={<button>Открыть галерею</button>}
  images={images}
  title="Заголовок галереи"
/>

// С начальным индексом (открыть сразу на 2-м изображении)
<GalleryDialog
  trigger={<button>Открыть</button>}
  images={images}
  initialIndex={1}
  title="Заголовок"
/>
```

## Props

| Prop | Тип | Обязательный | Описание |
|------|-----|--------------|----------|
| `trigger` | `ReactNode` | Да | Элемент, при клике на который открывается галерея |
| `images` | `GalleryImage[]` | Да | Массив изображений с `src` и опциональным `caption` |
| `title` | `string` | Нет | Заголовок галереи (для accessibility) |
| `initialIndex` | `number` | Нет | Индекс изображения, с которого открыть галерею (по умолчанию 0) |

## Где используется

1. **Секция "Как это работает"** — кнопка "Посмотреть примеры" открывает галерею с примерами записей
2. **Секция "Примеры записей с прибора"** — клик по любой карточке (с лупой при наведении) открывает галерею
